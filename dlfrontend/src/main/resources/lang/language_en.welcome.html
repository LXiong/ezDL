Welcome to ezDL

ezDL makes searching for documents in digital libraries easier, faster and more successful.

New features in version 1.4

  • Personal library with Mendeley synchronisation
  • Groupable result list
  • Support for wildcards in queries
  • Visualization of relations in search results
  • Easy combination of queries
  • Many other small improvements

Future releases will have ...

  • strategical and tactical support
  • support for multi-session searches
  • local search using your desktop search engine
  • even more digital libraries to search in

Please visit our web site to learn about feature for researchers and developers.
